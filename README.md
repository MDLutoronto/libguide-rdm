# HTML/CSS for LibGuide: Research Data Management

This is the active HTML/CSS code repository for the guide, **Research Data Managementa** (https://guides.library.utoronto.ca/rdm) that is saved on LibGuide via LibApps.

## How to use this repository
Update the relevant code and content changes in this repository to correspond with changes made on the LibGuide. 

## Tech stack

This LibGuide was written in the code editor VS Code. 

Changes can be committed on GitHub or VS Code depending on editor preference. 

## Reusing LibGuide content blocks
The assets developed in this guide are copyable via the internal U of T LibGuide, **UTL LibGuides Asset Library (Internal)**.
